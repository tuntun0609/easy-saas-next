由于 **Easy SaaS Next** 使用 [shadcn/ui](https://ui.shadcn.com/) 作为 UI 库。shadcn/ui 提供了丰富的组件，可以满足大部分场景。或者通过 [blocks](https://ui.shadcn.com/docs/blocks) 功能来添加开源社区的各种组件。

**Easy SaaS Next** 中使用了大量的开源组件，这里列一些常用的网站：

- https://magicui.design
- https://ui.aceternity.com
- https://tailark.com
- https://tweakcn.com
- https://originui.com
- https://shadcnui-blocks.com
- https://animate-ui.com/

一般只需要执行

```bash
pnpm dlx shadcn@latest add https://xxx/xxx.json
```

组件的源码就会自动添加到项目中。您就可以在项目中使用这些组件了。或者为其添加国际化支持
