## 编辑播客

你可以在 `content/blog` 目录下创建新的博客文章。

博客文章的格式如下：

```mdx
---
title: 博客文章标题
date: 2024-01-01
cover: image url
tags:
  - 标签1
  - 标签2	
---

播客内容

```

| 属性 | 说明 |
| --- | --- |
| title | 博客文章标题 |
| date | 博客文章发布日期 |
| cover | 博客文章封面图片 |
| tags | 博客文章标签 |
