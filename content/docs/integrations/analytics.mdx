目前支持的 Analytics 有：

- Google Analytics
- Umami
- Microsoft Clarity

您可以在 `.env` 文件中配置 Analytics 所需的信息，然后就可以将 Analytics 集成到您的网站中。

## Google Analytics

```bash
GA_ID=
```

## Umami

```bash
UMAMI_WEBSITE_ID=
UMAMI_BASE_URL=
```

## Microsoft Clarity

```bash
CLARITY_ID=
```

所有的组件位于 `src/components/analytics` 文件夹中。
