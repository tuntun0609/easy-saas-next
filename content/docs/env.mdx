### 参数说明：

| 参数 | 说明 |
|------|------|
| BASE_URL | 应用的基础URL地址 |
| USE_PROXY | 是否使用代理，可选值：true/false |
| ADMIN_USER_ID | 管理员用户ID，多个ID可用逗号分隔 |
| BETTER_AUTH_SECRET | Better Auth 认证密钥 |
| BETTER_AUTH_URL | Better Auth 服务地址（可选，默认为 BASE_URL） |
| POSTGRES_DATABASE_URL | PostgreSQL 数据库连接URL |
| GOOGLE_CLIENT_ID | Google OAuth 客户端ID |
| GOOGLE_CLIENT_SECRET | Google OAuth 客户端密钥 |
| GITHUB_CLIENT_ID | GitHub OAuth 客户端ID |
| GITHUB_CLIENT_SECRET | GitHub OAuth 客户端密钥 |
| UMAMI_WEBSITE_ID | Umami 网站统计ID |
| UMAMI_BASE_URL | Umami 服务基础URL |
| CLARITY_ID | Microsoft Clarity 分析ID |
| GA_ID | Google Analytics 分析ID |
| RESEND_API_KEY | Resend 邮件服务API密钥 |
| EMAIL_FROM_DOMAIN | 发送邮件的域名 |
| EMAIL_FROM_NAME | 发送邮件的显示名称 |
| CREEM_MODE | Creem支付模式，可选值：test/prod |
| CREEM_API_KEY | Creem API密钥 |
| CREEM_WEBHOOK_SECRET | Creem Webhook密钥 |
| NEXT_PUBLIC_CREEM_PRODUCT_ID | Creem产品ID（公开） |
