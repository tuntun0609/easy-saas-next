  ./.env.example

### Parameter Description:

| Parameter | Description |
|------|------|
| BASE_URL | Base URL of the application |
| USE_PROXY | Whether to use proxy, possible values: true/false |
| ADMIN_USER_ID | Administrator user ID, multiple IDs can be separated by commas |
| BETTER_AUTH_SECRET | Better Auth authentication key |
| BETTER_AUTH_URL | Better Auth service URL (optional, defaults to BASE_URL) |
| POSTGRES_DATABASE_URL | PostgreSQL database connection URL |
| GOOGLE_CLIENT_ID | Google OAuth client ID |
| GOOGLE_CLIENT_SECRET | Google OAuth client secret |
| GITHUB_CLIENT_ID | GitHub OAuth client ID |
| GITHUB_CLIENT_SECRET | GitHub OAuth client secret |
| UMAMI_WEBSITE_ID | Umami website analytics ID |
| UMAMI_BASE_URL | Umami service base URL |
| CLARITY_ID | Microsoft Clarity analytics ID |
| GA_ID | Google Analytics ID |
| RESEND_API_KEY | Resend email service API key |
| EMAIL_FROM_DOMAIN | Domain name for sending emails |
| EMAIL_FROM_NAME | Display name for sending emails |
| CREEM_MODE | Creem payment mode, possible values: test/prod |
| CREEM_API_KEY | Creem API key |
| CREEM_WEBHOOK_SECRET | Creem Webhook secret |
| NEXT_PUBLIC_CREEM_PRODUCT_ID | Creem product ID (public) |
